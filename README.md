# ember-inheritance-test

This project demonstrates how on a production build of Ember 3.13 it will fail to inherit methods from the parent class `Ember.Object` to a child class. 

To reproduce run `ember serve --environment=production`, visit localhost and press the `Say Hello` button. 

If run in a development environment Ember will successfully alert "Hello World". 

## Prerequisites

You will need the following things properly installed on your computer.

* [Git](https://git-scm.com/)
* [Node.js](https://nodejs.org/) (with npm)
* [Ember CLI](https://ember-cli.com/)
* [Google Chrome](https://google.com/chrome/)

## Installation

* `git clone <repository-url>` this repository
* `cd ember-inheritance-test`
* `npm install`
